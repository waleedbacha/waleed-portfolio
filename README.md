## 📱 About The Project

A modern, interactive, and fully responsive portfolio website showcasing my work as a Full Stack Developer. Built with React.js and featuring cutting-edge web technologies for an immersive user experience.

### 🎯 Key Features

- 🎨 **Dark/Light Mode** - Theme toggle with localStorage persistence
- ✨ **Custom Animated Cursor** - Interactive cursor with magnetic effects
- 🎭 **3D Elements** - Three.js rotating cube and interactive globe
- 🎪 **Particle Background** - Dynamic particle system with mouse interaction
- 📱 **Fully Responsive** - Optimized for all devices
- ⚡ **Smooth Animations** - Framer Motion powered transitions
- 📄 **PDF Resume Export** - Download professional resume
- 🎯 **Bento Grid Stats** - Modern statistics display
- 🔄 **Infinite Marquee** - Scrolling skills showcase
- 💬 **Toast Notifications** - User feedback system
- 🏆 **Achievement Badges** - Scroll-triggered gamification
- 📊 **Live Visitor Counter** - Real-time visitor tracking
- 🔗 **Share Portfolio** - One-click copy to clipboard

## 🛠️ Built With

- **Frontend Framework:** React.js 18
- **Styling:** CSS3, Bootstrap 5, Custom CSS Variables
- **Animations:** Framer Motion, Typed.js
- **3D Graphics:** Three.js, @react-three/fiber, @react-three/drei
- **Icons:** React Icons
- **PDF Generation:** html2pdf.js
- **Particles:** react-tsparticles, tsparticles-slim
- **Charts:** Chart.js
- **Email Service:** EmailJS
